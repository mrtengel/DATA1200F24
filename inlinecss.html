External style-sheets

This document has the <style> element inside the <head>. You should always have the CSS exclusively in a separate file (in a sub-folder). This is called an external stylesheet. The code for this is as follows, asuming the sub-folder is "style":
<link type="text/css" rel="stylesheet" href="style/main.css">
